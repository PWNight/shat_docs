Система создания и управления документацией для техникума, направленная на повышение удобства управления посещаемостью, создания справок и отчётов для студентов.

При инициализации приложения создайте файл .env и укажите в нём следующие переменные
MYSQL_SERVER="" # Адрес вашего mysql сервера

MYSQL_USER="" # Имя пользователя БД

MYSQL_PASSWORD="" # Пароль от пользователя

MYSQL_DATABASE="" # Название БД

SESSION_SECRET="" # Сгенерированный ключ, по которому будет шифроваться сессия

ROOT_EMAIL="" # Почта рут пользователя

ROOT_NAME="" # Имя рут пользователя

ROOT_PASSWORD="" # Пароль от рут пользователя
